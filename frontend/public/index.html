How Monte Carlo Control Works
① Generate Episode
Play a complete game using the current ε-greedy policy. Record every (state, action) pair encountered during play.
② Calculate Returns
At game end, the final reward (+1 win, −1 loss, 0 push) is propagated back through all visited (state, action) pairs.
③ Update Q-Values
Each first-visit pair updates its Q-value via incremental mean: Q ← Q + (R − Q) / N. Epsilon decays → exploitation increases.
